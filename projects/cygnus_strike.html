Cygnus Strike

A mobile game created for the Android platform.

Repository
About

Cygnus Strike is a mobile game developed collaboratively in Android Studio for my senior year highschool programming class. To make it easier on us, a Java game development framework called libGDX was used to support the application's creation. The game itself is based around the idea of a retro space shooter. The user controls a spaceship's position with one finger, while another finger is controlling the spaceship's shooting function. The goal of the game is pretty straightforward - prevent your hearts from dropping to zero by dodging or destroying the incoming enemy spacecrafts until the time runs out. Due to time constraints, a lot of the features originally envisioned (i.e., levels, shopping for upgrades, cross platform compatibility, improving general user enjoyment of gameplay) were regrettably not implemented.

Approach

The approach going into this project was very similar in style to what was done with PokéDodge. To start the project, a release schedule was defined before writing any code and follows roughly like this:

  1. Getting an image of a spaceship to follow user's finger
  2. Simulate "movement" by having the background scroll downwards
  3. Randomly spawn enemy space crafts
  4. Add unit collision to objects
  5. Add life and scoring system
  6. Add power ups
  7. Add win/lose conditions
  8. Implement screen navigation

Every three weeks, reports containing information such as current status, known bugs, and what's next would be created and given to the teacher. This would be followed by a sitdown meeting with discussions involving the contents of the report in addition to recommendations and feedback on the progress thus far.
