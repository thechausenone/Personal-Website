Cygnus Strike

A mobile game created for the Android platform.

Repository
About

Cygnus Strike is a mobile game developed collaboratively in Android Studio for my senior year highschool programming class. To support the aplication's creation, a Java game development framework, which included functionality to allow cross platform development, called libGDX was used. The game itself is based around the idea of a retro space shooter. The user controls a spaceship's position with one finger, while another finger is controlling the spaceship's shooting function. The goal of the game is pretty straightforward - prevent your hearts from dropping to zero by dodging or destroying the incoming enemy spacecrafts until the time runs out. Due to time constraints, a lot of the features originally envisioned (i.e., levels, shopping for upgrades, cross platform compatibility, improving general user enjoyment of gameplay) were regrettably not implemented.
